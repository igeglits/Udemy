Массив — сложная структура данных, которая хранит однотипные данных рядом и позволяют получить к ним доступ по индексу.

Создать массив в Java возможно двумя способами: с инициализацией значениями (такая инициализация указывается в фигурных скобках) и с указанием размера массива (в этом случае каждый элемент массива заполняется значением по умолчанию, которое зависит от типа данных массива).

Для вывода массива на консоль, рекомендуется использовать конструкцию Arrays.toString.

Индексация массива начинается с нуля!

Чтобы узнать длину массива, необходимо вызвать конструкцию array.length.

Ошибка ArrayIndexOutOfBoundsException возникает в случае выхода за границы массива.

Для итерации по массиву удобно использовать конструкцию foreach.

Массивы могут быть не только одномерными, но и многомерными: таблицы, кубы и т.д.
