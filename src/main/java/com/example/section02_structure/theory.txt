Операции цикла позволяют выполнять повторяющиеся действия в Вашей программе.

Операции цикла могут приводить к зависанию программы в случае если будет обнаружен бесконечный или цикл, требующий длительного времени на выполнение. В этом случае единственный способ остановить программу — это уничтожить процесс Java.

Отличайте бесконечные циклы от длительных операций. Возможно задача слишком сложная и программа не зависла, а требует длительного времени для ее корректного выполнения.

Если в алгоритме повторяющихся действий Вам нужен счетчик всегда используйте цикл for для этого.

Цикл while удобен в том случае когда счетчик итерации задан неявно.

Цикл do while встречается в реальных программах крайне редко и его всегда можно заменить на while.

Для создания бесконечных циклов удобен именно цикл while, поэтому рекомендуется использовать именно этот тип цикла для этих целей.

Бесконечные циклы удобны для описания алгоритмов для которых заранее не известно количество итераций.

В случае использования бесконечных циклов необходимо использовать оператор break, для их прерывания.

При правильном использовании break в конечных циклах, данный оператор должен использоваться только при условиях, которые не должны коррелировать с условиями самих циклов.

Оператор continue лучше избегать в программах. Какого то существенного выигрыша он не дает, но может запутать программиста при чтении чужой программы.

Используйте вложенные циклы для работы с многомерными структурами: таблица, куб и т.д.
